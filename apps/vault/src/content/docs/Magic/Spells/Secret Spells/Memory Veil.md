# Memory Veil


*Tier 3 Necrotic Spell*

**Concentration:** Ss long as the caster remains conscious.
**Effect:** Conceal or change all memories of an event for up to 12 creatures within 2 spaces on a failed WIL save for as long as this spell lasts.

> Why is this secret?
> This can have profound ethical and narrative implications. Often used by shadowy organizations or rulers to cover up secrets best left buried.
